# docs/workflow — Project tracking

Three files, updated every iteration:

| File | Purpose |
|---|---|
| [STATUS.md](STATUS.md) | Where we are right now: current iteration, next step, release % |
| [RELEASE-CHECKLIST.md](RELEASE-CHECKLIST.md) | Gate list before public GitHub release |
| [BENCHMARK-TARGETS.md](BENCHMARK-TARGETS.md) | Sandbox, target, and dogfood bench project specs |

**Rule:** update STATUS.md at the start and end of every iteration. RELEASE-CHECKLIST.md tracks the release gate — tick items as they close.
