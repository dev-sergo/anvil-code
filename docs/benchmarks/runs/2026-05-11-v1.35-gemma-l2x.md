## Code highlights

**L2.7** — Fastify query typing корректный:
```typescript
const { email } = request.query as { email?: string };
```

**L2.8** — module augmentation вместо прямого `request.startTime` (TS2339):
```typescript
declare module 'fastify' {
  interface FastifyRequest { startTime?: number; }
}
// + performance.now() в onRequest hook
```
Решает проблему которая блокировала qwen-coder на протяжении 10+ прогонов.

**L2.4** — хронический no-op на всех моделях. Rename refactor требует понимания
семантики "переименование метода", не "добавление функциональности". Отдельная проблема.
