## Root Cause Analysis

### 1. TestRunner 60s timeout (CRITICAL, fixed in v1.65c)

- Clean trpc test suite: **54 seconds**
- openapi codegen (triggered by any server/src/** change): **~7 seconds**
- Total during validation: **61 seconds → killed after 60s**

Every task that modifies `packages/server/src/**` triggers codegen cache miss → 7 extra seconds → total 61s → SIGKILL → "Tests failed (exit 1)" → commit_skipped.

**Fix**: increased TestRunner default from `60_000` to `120_000` ms.

### 2. Sandbox directory deletion

During the session, `/Users/admin/Documents/work/rag-system-sandbox` was accidentally deleted (likely from a git clean or worktree removal). A placeholder git repo was created to unblock the API. **Sandbox bench is non-functional until restored.**

### 3. tsx missing from pnpm store

`node_modules/.pnpm/tsx@4.19.3` was missing from the trpc pnpm store. Reinstalled via `pnpm install --filter @trpc/openapi`. Root cause unclear (possible pnpm store pruning).
