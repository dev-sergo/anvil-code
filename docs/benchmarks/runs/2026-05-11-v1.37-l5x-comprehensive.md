## Ceiling analysis

| Dimension | Success rate |
|---|---|
| 1-2 файла | ~100% |
| 3-4 файла | ~85% |
| 4-5 файла (new abstractions) | ~70% |
| 5+ файла (architectural) | ~30% |
| Algorithmic complexity | ~90% (logic bugs possible) |
| Real codebase vs synthetic | comparable when context fits |

**Quality ceiling:** Gemma generates correct TypeScript that compiles and runs for the vast majority of well-scoped tasks. Logic bugs (incorrect algorithms) occur ~10-20% and require test coverage to catch. Architectural multi-file changes beyond 4 files hit Reviewer or validation limits.
